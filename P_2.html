Data Cleaning¶

  • Replacing all Na to Unknown for better interpretation of the dataset
  • Casting release_date to date instead of string datatype.
  • Casting budget, revenue, budget_adj, and revenue_adj from float to int datatype for a better representation.
  • Changing all 0 values from budget, revenue, budget_adj, and revenue_adj to prevent distorting results.
  • Remove all unnecessary columns to make the investigation easier.

Convert the following to proper datatypes:

df['budget_adj'] = df['budget_adj'].astype(int)
df['revenue_adj'] = df['revenue_adj'].astype(int)
